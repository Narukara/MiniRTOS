## A Minimal RTOS

Compatible with ARMv7-M architecture. Main features:

- Create tasks
- Simple round robin & preemptive scheduling
- Delay tasks

Tested on STM32F103

---

## 简单的 RTOS 内核

适用于 ARMv7-M 架构，实现的功能:

1. 创建任务
2. 简单的时间片轮转 & 抢占式调度
3. 使任务延时

在 STM32F103 上进行了测试

主要参考了《ARM Cortex-M3 与 Cortex-M4 权威指南》
